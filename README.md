# 세컨블록 블록 관리 툴

## 세컨블록

### 개요
[세컨블록](https://2ndblock.com)은 한국의 가상자산 거래소인 [업비트](https://upbit.com)의 운영사 두나무에서 운영중인 메타버스 플랫폼이다.  
픽셀 감성이 묻어나는 나만의 공간에서 나의 개성과 취향이 담긴 캐릭터를 이용해 다양한 사람들과 더 즐겁고 특별하게 소통할 수 있는 공간이다.
